## Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS
- **Backend**: Appwrite (Database, Auth, Storage)
- **Payment**: Vipps MobilePay
- **i18n**: next-intl (Norwegian & English)
- **Monorepo**: Turborepo with Bun
